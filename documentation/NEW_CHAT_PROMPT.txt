👋 Hej! Jeg er klar til at deploye Rekruna CV Screener til production.

## 📋 Projekt Status

**Platform**: Next.js 14 CV screening SaaS med AI (GPT-4)
**Tech Stack**: Next.js 14, Supabase, Stripe, OpenAI, React-PDF
**Status**: ✅ Fully tested i local miljø - klar til deployment

## 🎯 Hvad virker (local test completed):

1. ✅ Authentication (Supabase Auth)
2. ✅ Job upload (PDF) → AI extracts 2-5 requirements
3. ✅ CV upload (PDF, op til 50 stk.)
4. ✅ AI Analysis (GPT-4) → Scoring + ranking
5. ✅ PDF Report generation (med Requirements Legend: A, B, C, D design)
6. ✅ Compare Mode (sammenlign flere analyser af samme job)
7. ✅ Job Templates (gem/genbrug job beskrivelser)
8. ✅ Credit System (Pay-as-you-go + subscriptions)
9. ✅ Stripe Integration (checkout + webhooks)
10. ✅ User Profiles (30 dage history)
11. ✅ Custom Requirements (hvis AI extraction fejler)
12. ✅ Error handling med Toast notifications (dansk)
13. ✅ Responsive design (tablet/mobile)
14. ✅ Singleton Supabase client (no auth conflicts)

## 🗄️ Database Setup Required:

Følgende migrations skal køres i Supabase (ligger i `database_migrations/`):
1. `add_user_profiles_table.sql`
2. `add_credit_system_complete.sql`
3. `ensure_analysis_results_table.sql`
4. `add_cache_tables.sql`
5. `add_job_templates_table.sql`

Storage bucket skal oprettes: `analysis-reports` (private, PDF only)

## 💳 Stripe Setup Needed:

3 produkter skal oprettes:
1. **Pay As You Go**: 99 DKK per analyse
2. **Pro Plan**: 699 DKK/md (10 credits)
3. **Business Plan**: 1499 DKK/md (25 credits)

Webhook events: `checkout.session.completed`, `customer.subscription.*`, `invoice.payment_*`

**VIGTIGT**: Product IDs skal opdateres i:
- `app/api/checkout/route.ts`
- `app/api/webhooks/route.ts`
